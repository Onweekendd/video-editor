# EaseClip

EaseClip 是一款功能强大的在线视频编辑器，提供专业的视频编辑功能，让创作变得轻松自在。

## 🌟 核心功能

### 📥 视频导入与播放

- **支持格式**
  - ⭕ 视频格式：MP4, AVI, MOV, WMV, FLV
  - ⭕ 分辨率支持：4K, 1080p, 720p
  - ⭕ 帧率支持：24fps, 30fps, 60fps
- **导入方式**
  - ⭕ 拖拽导入
  - ⭕ 本地文件选择
  - ⭕ 批量导入
- **播放控制**
  - ⭕ 基础播放控制
  - ⭕ 帧级控制
  - ⭕ 速度调节(0.25x - 2x)
  - ⭕ 循环播放

### ⏱️ 时间线系统

- **可视化时间轴**
  - ⭕ 灵活的时间刻度
  - ⭕ 时间轴缩放
  - ⭕ 关键帧标记
- **多轨道系统**
  - ⭕ 多视频轨道
  - ⭕ 独立音频轨道
  - ⭕ 文字特效轨道
  - ⭕ 图片素材轨道
- **时间线操作**
  - ⭕ 拖拽调整
  - ⭕ 快捷菜单
  - ⭕ 精确定位
  - ⭕ 关键帧标记
